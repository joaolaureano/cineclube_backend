# Cine Clube - Backend

> Descrição do projeto

## Pré-requesitos

- Node.js
- npm

## Iniciando

- Clonar este repositório  
`git clone http://tools.ages.pucrs.br/cine-clube/backend.git`

- Entrar no diretório do projeto  
`cd backend`

- Instalar as dependências  
`npm install`

- Iniciar a aplicação  
`npm run dev`

- Acesse pelo navegador  
`http://localhost:5000`
